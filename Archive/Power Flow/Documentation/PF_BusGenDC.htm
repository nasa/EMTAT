Power Flow DC Generator Bus

 

Block Description

This block acts to output a constant user specified power by adjusting either a voltage or current input signal for the appropriate power output in simulation.

Mask Parameters

PeInEN_M External Pelec
P_M generated real power

Input Variables

Vin Voltage at input
I Current entering node through input.

Output Variables

Vout Voltage at output
P_Er Difference between calculated power and specified power. Dependent to be driven to zero by the local solver.
Data Calculated power
