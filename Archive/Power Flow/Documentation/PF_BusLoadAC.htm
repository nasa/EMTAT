Power Flow DC Load Bus

 

Block Description

The load bus component enables the addition of external known power loads. For this block, a complex power load is defined. This complex power is then compared with a calculated complex power input to determine a power error that can be used to back calculate the system voltages. The basic equations that govern the block are shown in the equation below. Note: P and Q (the bussed signals contained in Data) are calculated values, P_M and Q_M are known values, and P_er and Q_er are dependents to be driven to zero by the solver

Vin = Vout

S = Vin × ∑I*           (Note that I* is the conjugate of complex current)
 
P_Er = P - P_M
 
Q_Er = Q - Q_M
 

Mask Parameters

PeInEN_M Externally defined real power
P_M User defined constant real power
QeInEN__M Externally defined reactive Power
Q_M User defined constant reactive power

 

Input Variables

Vin Voltage at side A. It can be a scalar value for a real voltage or a vector for a phasor voltage where the first element is magnitude and the second is phase angle
I_comp Complex current through node

Output Variables

Vout Voltage at side B
P_Er difference between calculated real power and specified real power P_M or PeInEN_M. This is a solver dependent variable to be driven to zero by the solver.
Q_Er difference between calculated real power and specified real power Q_M or QeInEN_M. This is a solver dependent variable to be driven to zero by the solver.
Data Bussed output consisting of calculated P (Real Power) and Q (Reactive Power)
