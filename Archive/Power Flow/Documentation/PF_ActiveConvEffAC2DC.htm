Power Flow AC to DC Converter

 

Block Description

This block converts AC to DC for connection between AC based powerflow components and DC based powerflow components. The phase angle input must be in radians and works best a constant, as the Newton Raphson solver frequently cannot force associated power error outputs to zero when using a phase angle as an independent variable due to the angle's rotational properties. 

Mask Parameters

Eff_M Converter Efficiency
EffInEN_M Externally Defined Efficiency

Input Variables

VA Input phasor voltage: the first element is magnitude and the second is phase angle
VBdc Desired output DC voltage
IAg Input current
I_phiAg Phase angle of the input current (radians)

Output Variables

VB Output DC voltage
IB Output DC current
IcA Complex (AC) input current
