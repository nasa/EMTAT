Power Flow DC Load Bus

 

Block Description

The load bus component enables the addition of external known power loads. For this block, a real power load is defined. This real power is then compared with a calculated real power input to determine a power error that can be used to back calculate the system voltages. The basic equations that govern the block are shown in the equation below. Note: P (the bussed signal contained in Data) is a calculated value, P_M is a known value, and P_er is a dependent to be driven to zero by the solver.

Vin = Vout

P = Vin × ∑I         
 
P_Er = P - P_M
