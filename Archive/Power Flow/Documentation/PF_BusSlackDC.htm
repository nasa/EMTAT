Power Flow DC Slack Bus

 

Block Description

The slack bus component offers a general purpose starting point to the model that will account for any power disparities in the system. This block acts as a way of specifying the voltage of the system, and uses the summation of all input and output currents to determine a real power.

Vin = Vout

Data (Power Output) = P

                    P = Vin × ∑I

Mask Parameters

N/A

Input Variables

Vin Voltage at input. Usually a magnitude of 1 PU (per unit). Used in conjunction with current input to calculate power demand on the bus.
I Current flowing through output.

Output Variables

Vout Same value as Vin. Connects to voltage input of downstream component
Data Power demand on bus. (Demand of downstream network)
