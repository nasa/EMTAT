Temperature (Ploss)

Block Inputs

Ploss - Power Loss, W (n x 1)
Tamb - Ambient temperature, degF (n x 1)
Qc - Heat due to cooling, btu (n x 1)

Block Outputs

Temp - temperature of component(s), degF (n x 1)

Block Mask

m - mass, lbm (n x 1)
Cp - Specific heat, btu/(lbm-degF) (n x 1)
Qo - Initial heat from electrical component, btu (n x 1)

Block Calculations

General Variables

t - Time, sec
n - Size of Input/Output Vector
