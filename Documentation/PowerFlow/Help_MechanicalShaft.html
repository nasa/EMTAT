Mechanical Shaft

Block Inputs

T - Torque, lbf-ft (n x 1)
Fail? - Failure Toggle, 0 = Normal Operation/1 = Failure (n x 1)
N - Rotational Velocity, rpm (n x 1)

Block Outputs

Outputs - Bus of Block Inner Variables
Tnet - Net Torque, lbf-ft (n x 1)
N - Rotational Velocity, rpm (n x 1)
Ndot - Rotational Acceleration, rpm/sec (n x 1)
Q - Heat generated by friction, btu (n x 1)
SBFail - Shaft Break Failure, 0 = No Failure/1 = Failure (n x 1)
Heat - Heat generated by friction, btu (n x 1)
Ndot - Rotational Acceleration, rpm/sec (n x 1)

Block Mask

J - Rotational Inertia, lbf-ft-s^2 or slugs-ft^2 (n x 1)
B - Rotational Viscous Damping Coefficent, lbf-ft-s (n x 1)

Block Calculations

Tnet = (T - (B * N * (2*pi)/60))
Ndot = (60/(2*pi)) * Tnet/J)

General Variables

n - Size of Input/Output Vector
