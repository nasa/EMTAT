Electrical Bus

Block Inputs

I - Direct Current, A (n x 1)
V - Bus Voltage, V (n x 1)

Block Outputs

V - Bus Voltage, V (n x 1)
Vdot - Bus Voltage Derivative, V/sec (n x 1)

Block Mask

C - Capacitance, F (n x 1)
Nb - # of branches (1 x 1)

Block Calculations

Vdot = I/C
V = V

General Variables

n - Size of Input/Output Vector
