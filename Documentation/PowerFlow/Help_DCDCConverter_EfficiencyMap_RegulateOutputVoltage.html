DC-DC Converter (Efficiency Map) (Regulate Output Voltage)

Block Inputs

V_2 - Output or Load Voltage, V (n x 1)
V_1 - Input or Source Voltage, V (n x 1)
I*_2 - Output or Load Setpoint Current, A (n x 1)
Fail? - Failure Toggle, 0 = Normal Operation/1 = Failure (n x 1)
Deg - Percent Degredation, 0 = No Degredation to 1 = Full Degredation, (n x 1)

Block Outputs

Outputs - Bus of Block Inner Variables
P_1 - Input or Source Power, W (n x 1)
P_2 - Output or Load Power, W (n x 1)
Eff - Efficiency (n x 1)
V_2 - Output or Load Voltage, V (n x 1)
Vdot_2 - Output or Load Voltage Derivative, V/sec (n x 1)
V_1 - Input or Source Voltage, V (n x 1)
Vdot_1 - Input or Source Voltage Derivative, V/sec (n x 1)
I_1 - Input or Source Current, A (n x 1)
Idot_1 - Input or Source Current Derivative, A/sec (n x 1)
I_2 - Output or Load Current, A (n x 1)
Idot_2 - Output or Load Current Derivative, A/sec (n x 1)
Ploss - Power Loss, W (n x 1)
I*_2 - Output or Load Setpoint Current, A (n x 1)
Ploss - Power Loss, W (n x 1)
I_1 - Output or Load Voltage, A (n x 1)
I_2 - Input or Source Current, A (n x 1)

Block Mask

X_Vin_VoltageVec_M - Input/Source Voltage (V_1) Vector, V (m x 1)
Y_Iout CurrentVec_M - Output/Load Current (I_2) Vector, A (p x 1)
T_C_EffArray_M - Efficiency Matrix, W (m x p)
Mode - Faulure Mode Selector, 1 - Open Circuit Failure/2 - Degredation Failure (n x 1)

Block Calculations

General Variables

n - Size of Input/Output Vector
m - Size of Efficiency Table Input/Source Voltage (V_1) Vector
p - Size of Efficiency Table Output/Load Current (I_2) Vector
