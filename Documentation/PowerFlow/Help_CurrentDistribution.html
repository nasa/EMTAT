Current Distribution

Block Inputs

I - Vector of Supply Currents, A (n x 1)
Idmd - Vector of Current Demands, A (n x 1)

Block Outputs

Isplit - Vector of Currents Split Based Upon Demand, A (n x 1)

Block Mask

Block Calculations

Case where n = m:
Example:
Isplit(1) = 1*4/sum(4,5,6) = 0.2667
Isplit(2) = 2*5/sum(4,5,6) = 0.6667
Isplit(3) = 3*6/sum(4,5,6) = 1.2
Case where n < m:
end
Example:
Isplit(1:2) = 1*[3;4]/sum(3,4) = [0.4286; 0.5714]
Isplit(3:4) = 1*[5;6]/sum(5,6) = [0.9091; 1.091]

General Variables

n - Size of I Vector
m = Size of Idmd Vector
i = Index of I Vector
