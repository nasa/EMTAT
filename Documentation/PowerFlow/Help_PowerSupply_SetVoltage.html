Power Supply (Set Voltage)

Block Inputs

V* = Voltage Setpoint, V (n x 1)
I - Current, A (n x 1)

Block Outputs

Outputs - Bus of Block Inner Variables
V - Voltage, V (n x 1)
Isum - Summation of currents per branch, A (n x 1)
P = Electrical Power, W (n x 1)
V - Voltage, V (n x 1)

Block Mask

Nb - # of branches (1 x 1)

Block Calculations

V = V*
P = Isum * V*

General Variables

n - Size of Input/Output Vector
