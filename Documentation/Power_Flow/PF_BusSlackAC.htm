Power Flow AC Slack Bus

 

Block Description

The slack bus component offers a general purpose starting point to the model that will account for any power disparities in the system. This block acts as a way of specifying the voltage of the system, and uses the summation of all input and output currents to determine a complex power. The equations describing the block behavior are as follows:

Vin = Vout

Data (Complex Power Output) = [P,Q]

                    P = Re{Vin × ∑I*}
                    Q = Im{Vin × ∑I*}         
 

Mask Parameters

N/A

Input Variables

Vin Voltage at side A. Usually a vector whose first element is a voltage magnitude of 1 PU (per unit) and whose second element is a phase angle of 0. Used in conjunction with current input to calculate power demand on the bus.
I_cmp Complex current flowing through side B.

Output Variables

Vout Same value as Vin. Connects to voltage input of downstream component
Data Bussed signal consisting of the real(P) and imaginary(Q) components of the bus power.
