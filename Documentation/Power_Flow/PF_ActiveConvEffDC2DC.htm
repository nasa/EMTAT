Power Flow DC to DC Converter

 

Block Description

This block converts DC voltages from one level to another, taking into account efficiency losses.

Mask Parameters

Eff_M Converter Efficiency
EffInEN_M Externally Defined Efficiency

Input Variables

VA Input DC Voltage
VB Desired output DC voltage
Iag Input current

Output Variables

VB Ouput DC voltage
IB Output current
IA Input current
